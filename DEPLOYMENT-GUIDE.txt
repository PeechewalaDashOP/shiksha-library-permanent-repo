## STEP 1 — SET UP SUPABASE (Free Database + Auth)
Time: ~10 minutes

1. Go to https://supabase.com → Click "Start your project"
2. Sign up with GitHub or email
3. Click "New Project" → Fill in:
   - Project name: shiksha-library
   - Database Password: (save this somewhere safe)
   - Region: South Asia (ap-south-1)
4. Wait ~2 minutes for project to be ready
5. In left sidebar → Click "SQL Editor"
6. Click "New Query"
7. Open the file: supabase-setup.sql
8. Copy ALL the text → Paste in the SQL editor
9. Click "Run" (green button)
10. You should see: "Success. No rows returned"

11. Now get your keys:
    - Left sidebar → Settings → API
    - Copy "Project URL" → save as SUPABASE_URL
    - Copy "anon public" key → save as SUPABASE_ANON_KEY
    - Copy "service_role secret" key → save as SUPABASE_SERVICE_ROLE_KEY
    ⚠️ NEVER share the service_role key publicly

---

## STEP 2 — SET UP RAZORPAY (Free Payment Gateway)
Time: ~10 minutes

1. Go to https://razorpay.com → Sign Up
2. Fill business details:
   - Business type: Individual / Proprietorship
   - Use your own PAN / Aadhaar
3. Go to Settings → API Keys → Generate Test Key
4. Copy:
   - Key ID → save as RAZORPAY_KEY_ID
   - Key Secret → save as RAZORPAY_KEY_SECRET
5. ⚠️ Test mode first! After everything works, switch to Live keys.

## STEP 4 — ADD YOUR KEYS TO ALL FILES

## STEP 5 — ADD ENVIRONMENT VARIABLES TO NETLIFY
Time: ~3 minutes
(This is for the backend functions — keeps secret keys safe)

1. Go to https://app.netlify.com → Your site
2. Site configuration → Environment variables → Add variable

Add these 4 variables:
  Name: RAZORPAY_KEY_ID       Value: (your Razorpay key ID)
  Name: RAZORPAY_KEY_SECRET   Value: (your Razorpay key secret)
  Name: SUPABASE_URL          Value: (your Supabase URL)
  Name: SUPABASE_SERVICE_ROLE_KEY  Value: (your Supabase service role key)

Click Save.

---

## STEP 6 — ADD SCRIPTS TO YOUR index.html
Time: ~5 minutes

Open your existing index.html file.
Find the </body> tag near the bottom.
Just BEFORE </body>, add these 3 lines:

  <script src="https://checkout.razorpay.com/v1/checkout.js"></script>
  <script src="https://cdn.jsdelivr.net/npm/@supabase/supabase-js@2"></script>
  <script src="js/booking.js"></script>

That's it! The booking system attaches automatically to all "Book Now" buttons.

## COMMON ISSUES & FIXES

Q: "Book Now" button still opens WhatsApp
A: Check that booking.js script tag is added before </body> in index.html

Q: Payment window doesn't open
A: Check Razorpay key in booking.js is correct. Check browser console for errors.

Q: "Verification failed" after payment
A: Check Netlify environment variables are set correctly (Step 5)

Q: Can't login to dashboard
A: Default password = phone number entered during registration

Q: Admin dashboard shows no data
A: Make sure ADMIN_EMAIL in admin.html matches your Supabase login email exactly
